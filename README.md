# Numo::Random

Numo::Random provides random number generation with several distributions for Numo::NArray.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'numo-random'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install numo-random
